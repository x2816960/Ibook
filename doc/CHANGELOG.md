# Ibook 更新日志

所有版本更新记录。遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [1.3.0] - 2026-03-23

### 安全修复
- 附件下载接口恢复认证验证：防止用户通过猜测 attachment ID 访问他人的附件
- 新增 `get_current_user_optional` 支持从 query 参数读取 token，解决 Markdown 预览图片/视频的认证问题
- 前端 Markdown 渲染时自动给图片 URL 附加 token

### 优化
- 附件存储路径优化：`{user_id}/{task_id}/{yyyy-mm-dd}_{uuid}.{ext}`
  - 保留 task_id 层级，便于按任务管理文件
  - 文件名增加日期前缀，一眼识别上传时间
  - 随机 UUID 命名，避免文件名冲突，隐藏原始文件名

### Bug 修复
- 修复永久删除任务时附件文件未同步删除的问题

## [1.2.0] - 2026-03-23

### Bug 修复
- 修复 Markdown 预览中图片无法显示的问题：附件下载 URL 不再包含 token，避免用户重新登录后 token 过期导致图片 401 错误

### 优化
- 附件下载接口改为无需认证，简化 URL 格式
- Markdown 编辑器和预览组件清理冗余代码

## [1.1.0] - 2026-03-21

### 新增
- 备份删除 API：支持删除单个备份文件和一键删除所有备份
- 前端备份管理界面：每行显示删除按钮，备份历史卡片头部显示一键删除所有按钮

### Bug 修复
- 修复备份下载返回 "Not authenticated" 错误：改用 fetch + blob 方式下载
- 修复任务拖动排序后弹出错误提示：添加顺序变化检查
- 修复创建第3个任务时无法保存：创建任务后关闭抽屉
- 修复导入备份后刷新界面看不到数据：导入时先关闭所有数据库连接池，再删除旧WAL文件，最后复制新数据库

## [1.0.0] - 2026-03-21

### 新增
- 用户注册/登录系统，JWT 认证，bcrypt 密码加密
- 管理员账户，首次部署交互式设置密码
- 账户锁定机制（5次失败锁定15分钟），管理员可手动解锁
- JWT 退出登录黑名单机制
- 任务 CRUD（创建、编辑、删除、查看）
- 任务状态管理（待办/进行中/已完成/已取消），支持已完成回退到进行中
- 任务优先级（高/中/低）
- 任务截止时间，支持无限期选项
- Markdown 详细信息编辑器（md-editor-v3）
- 附件上传（图片/视频/文件），统一通过附件表存储
- 图片点击放大查看，视频在线播放，附件下载
- 任务筛选（状态、优先级）和关键词搜索
- 拖拽排序，排序持久化
- 分页展示（10/20/50条）
- 任务统计面板（总数、各状态、今日到期、已过期）
- 回收站（软删除、恢复、永久删除、30天自动清理）
- 个人设置（修改昵称、修改密码）
- 管理员用户管理（查看、禁用/启用、解锁、重置密码）
- 管理员系统配置（附件大小限制）
- 管理员系统统计
- 数据备份导出（数据库+附件打包 tar.gz）
- 数据备份导入恢复
- CLI 备份/恢复脚本
- Ubuntu 一键安装脚本（支持 20.04/22.04/24.04）
- Docker 部署方案
- 项目文档（README、API、部署说明）
